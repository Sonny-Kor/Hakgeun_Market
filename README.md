# Hakgeun_Market (학근 마켓)
<hr/>

## 개요    

학생들을 위한 중고 거래 서비스      
금오공과대학교 - 오픈소스 프로젝트     
<hr/>

## 구현 기능    

 회원관리     
 중고거래 관리 (상품 게시글 CRUD)     
 거래를 위한 채팅 기능     
<hr/>
 
## 🛠️ 개발 환경     

 UI: Fultter     
 DB: FireBase
